# 📱 RNS Bluetooth Messenger

Android application for sending text and images over the Reticulum Network Stack via Bluetooth Classic to RNode devices.

## Features

- ✅ Bluetooth Classic (SPP/RFCOMM) connection to RNode
- ✅ Text message sending/receiving
- ✅ Image transfer with compression
- ✅ Reticulum Network Stack integration
- ✅ Byte-perfect interoperability with Python RNS
- ✅ Background service for persistent connection

## Requirements

- Android 8.0+ (API 26)
- Bluetooth Classic support
- Paired RNode device
- JDK 21+
